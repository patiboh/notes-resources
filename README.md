# Serverless Resources for Notes App

Based on the [Serverless Stack](http://serverless-stack.com) guide.
This repo follows the [Best Practices Section](https://serverless-stack.com/chapters/best-practices-for-building-serverless-apps.html), and it is dedicated to the project's shared resources (AWS Infrastructure).

Original starter code & README can be found here: [Serverless Stack Demo Extended - Resources](https://github.com/AnomalyInnovations/serverless-stack-demo-ext-resources)

---

## Usage

To use this repo locally you need to have the [Serverless framework](https://serverless.com) installed.

```bash
$ npm install serverless -g
```

Clone this repo.

```bash
$ git clone https://github.com/AnomalyInnovations/serverless-stack-demo-ext-api
```

Go to one of the services in the `services/` dir.

And run this to deploy to your AWS account.

```bash
$ serverless deploy
```
